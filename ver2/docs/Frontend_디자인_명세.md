| `white` | `#FFFFFF` | 주요 텍스트 |
| 상태: success | `#22C55E` | 승인/가능 |
| 상태: error | `#EF4444` | 거절/오류 |

| 용도 | Font | Weight |
|------|------|--------|

| 본문 (한글) | Noto Sans KR | 400, 500, 700 |

## 1. 공통 컴포넌트

### 1.1 Header
```
┌────────────────────────────────────────────────────┐
│ [Logo: We are Live]     [🔍] [🔔 badge] [Avatar ▾] │
└────────────────────────────────────────────────────┘
```
- 로고: Bebas Neue, accent 컬러
- 알림 벨: 읽지 않은 알림 수 badge (accent 배경)
- 프로필 드롭다운: 마이페이지, 내 밴드, 내 조직, 로그아웃

### 1.2 Bottom Navigation (모바일)
```
┌──────────────────────────────────┐
│  🏠 홈  │  🎵 탐색  │  🏛 공연장  │  👤 마이  │
└──────────────────────────────────┘
```
- 4탭 고정
- 활성 탭: accent 컬러 + filled icon
- 비활성: muted 컬러 + outline icon

### 1.3 카드 컴포넌트
- 배경: surface-card
- border-radius: 12px
- border: 1px solid border
- hover: surface-elevated 전환, 미세 scale(1.02)
- 그림자: 없음 (다크 테마)

### 1.4 뱃지/태그
- 장르 태그: surface-elevated 배경, subtle 텍스트, border-radius: 9999px, px-3 py-1
- 상태 뱃지:
  - PENDING: warning 배경 (반투명)
  - APPROVED/AVAILABLE: success 배경 (반투명)
  - REJECTED/BLOCKED: error 배경 (반투명)
  - CANCELLED: muted 배경

### 1.5 버튼
| 종류 | 배경 | 텍스트 | hover |
|------|------|--------|-------|
| Primary | accent | black | accent-hover |
| Secondary | surface-elevated | white | border 강조 |
| Ghost | transparent | subtle | surface-elevated |
| Danger | error (반투명) | error | error (진하게) |

## 2. 화면별 상세 명세

### 2.1 홈 (`/`)

**목적:** 다가오는 공연 목록 중심의 랜딩 페이지

```
┌─────────────────────────────────────────┐
│ [Header]                                 │
├─────────────────────────────────────────┤
│                                          │
│  🔥 다가오는 공연                         │
│  ┌──────┐ ┌──────┐ ┌──────┐             │
│  │poster│ │poster│ │poster│  → 가로스크롤 │
│  │title │ │title │ │title │             │
│  │date  │ │date  │ │date  │             │
│  │venue │ │venue │ │venue │             │
│  └──────┘ └──────┘ └──────┘             │
│                                          │
│  🎵 추천 밴드                             │
│  (○)(○)(○)(○)(○)  → 원형 프로필 가로스크롤  │
│                                          │
│  📋 전체 공연 목록                         │
│  ┌─────────────────────────────────┐    │
│  │ [포스터] 공연명 / 날짜 / 공연장     │    │
│  ├─────────────────────────────────┤    │
│  │ [포스터] 공연명 / 날짜 / 공연장     │    │
│  └─────────────────────────────────┘    │
│                                          │
│ [Bottom Nav]                             │
└─────────────────────────────────────────┘
```

**데이터 매핑 (Performance):**
- posterImage → 포스터 썸네일
- title → 공연명
- date + startTime → 날짜/시간
- venue.name → 공연장명
- status → 상태 뱃지
- ticketPrice → 가격 (0이면 "무료")

## 5. 디자인 터미널 작업 요청사항

1. **공연장 상세** 페이지 리디자인: `amenities[]`, `operatingHours JSON`, `rentalFee JSON`, `lat/lng 지도` 반영
2. **예약 폼** 리디자인: `eventType`, `expectedSize`, `startTime/endTime` required 반영
3. **밴드 카드/상세**: `coverImage`, `status`, `inviteCode` 반영
4. **조직 상세**: `coverImage`, `school`, `region`, `snsLinks`, `isPinned` 공지 반영
5. **공연 등록 폼**: `startTime/endTime`, `setlist` 편집 UI 추가
6. **마이페이지**: `bio` 필드 표시/편집 추가
7. **밴드 가입 페이지**: 초대코드 입력 화면 신규 생성
